# Nginx
> Nginx是非同步框架的網頁伺服器，也可以用作反向代理、負載平衡器和HTTP快取。
>該軟體由伊戈爾·賽索耶夫建立並於2004年首次公開發布。
>2011年成立同名公司以提供支援。2019年3月11日，Nginx公司被F5 Networks以6.7億美元收購。
> Nginx是免費的開源軟體，根據類BSD授權條款的條款釋出。
